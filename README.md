## Dataset Description

The dataset contains e-commerce order information, including:

* Order ID
* Customer ID
* Product Category
* Product Price
* Discount Percentage
* Delivery Duration
* Customer Rating
* Return Status (Returned / Not Returned)

## Project Objectives

* Understand return behavior patterns.
* Analyze factors affecting product returns.
* Identify high-risk product categories.
* Investigate the impact of delivery delays.
* Perform statistical hypothesis testing.
* Segment customers based on return behavior.
* Provide business recommendations to reduce returns.

### Descriptive Statistics

* Mean
* Median
* Standard Deviation
* Quartiles
* Distribution Analysis

### Return Pattern Analysis

* Category vs Return Rate
* Price vs Return Rate
* Delivery Time vs Return Rate
* Discount vs Return Rate
* Customer Rating vs Return Rate

### Outlier Detection

* Box Plot Analysis
* Interquartile Range (IQR)
* Z-Score Method

### Hypothesis Testing

**Null Hypothesis (H₀):**
Delivery delay has no impact on product return behavior.

**Alternative Hypothesis (H₁):**
Delivery delay significantly impacts product return behavior.

Test Used:

* Independent T-Test

### Customer Segmentation

Customers were categorized into:

* High Return Customers
* Discount Driven Customers
* Premium Customers
* Delayed Delivery Customers
* Low Rating Customers

## Key Findings

### 1. Product Categories

Certain categories showed significantly higher return rates compared to others.

### 2. Delivery Delays

Orders with longer delivery durations were more likely to be returned.

### 3. Product Pricing

High-priced products exhibited a higher probability of returns.

### 4. Discounts

Large discounts attracted more purchases but were associated with increased return rates.

### 5. Customer Ratings

Lower customer ratings were strongly linked with returned products.
